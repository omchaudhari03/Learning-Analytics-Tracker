# Learning Analytics Tracker

While learning programming and web development, I often found it difficult to understand whether I was actually making progress or simply completing tutorials. I wanted a simple way to track skill growth, learning consistency, and personal goals in one place.

Learning Analytics Tracker is a Java-based application that helps users monitor their learning journey through skill tracking, goal management, achievement tracking, and performance insights. The application allows users to track progress across multiple skills, monitor study activity, set learning goals, identify strengths and weaker areas, generate progress reports, and celebrate learning milestones.

I built this project to apply Object-Oriented Programming concepts to a real problem I personally faced as a student while developing a better understanding of software design and problem-solving.

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework
