# Learning-Analytics-Tracker


While learning programming and web development, I often found it difficult to keep track of what I had actually learned and where I was making progress. Simply completing tutorials or courses did not provide a clear picture of consistency, skill growth, or learning gaps.

To address this, I built Learning Analytics Tracker, a Java-based application that helps users monitor their learning journey through skill tracking, goal management, and performance analytics.

## Features

* Track progress across multiple skills
* Monitor study consistency and learning activity
* Set learning goals and track completion
* Identify strongest and weakest skills
* Generate progress reports and learning insights
* Track achievements and milestones

## Why I Built This

As a first-year Computer Science student, I wanted a simple way to measure my learning progress beyond grades and course completion. This project helped me apply object-oriented programming concepts to solve a problem I personally faced while learning new technologies.

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework

## Key Concepts Applied

* Classes and Objects
* Encapsulation
* Modular Design
* Analytics Logic
* Report Generation
